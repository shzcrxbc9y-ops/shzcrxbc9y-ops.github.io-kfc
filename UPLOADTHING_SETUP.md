# Настройка UploadThing

## Шаги для настройки:

### 1. Получите UPLOADTHING_APP_ID:
   - Войдите в дашборд UploadThing: https://uploadthing.com
   - Перейдите в настройки вашего приложения
   - Скопируйте `UPLOADTHING_APP_ID` (обычно начинается с `app_`)

### 2. Добавьте переменные окружения в Vercel:

**Шаги:**
1. Перейдите на https://vercel.com
2. Выберите ваш проект
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте следующие переменные:
   - **Name:** `UPLOADTHING_TOKEN`
     **Value:** `ваш_секретный_ключ_из_uploadthing` (начинается с `sk_live_...`)
   - **Name:** `UPLOADTHING_APP_ID` (опционально, если требуется)
     **Value:** `ваш_app_id_из_дашборда_uploadthing` (если требуется)

5. Выберите **Production**, **Preview**, и **Development** для обеих переменных
6. Нажмите **Save**

**Важно:** 
- Используйте `UPLOADTHING_TOKEN` (не `UPLOADTHING_SECRET`)
- **НЕ добавляйте** секретные ключи в файлы, которые коммитятся в git

### 3. Перезапустите деплой:
   - После добавления переменных Vercel автоматически перезапустит деплой
   - Или перейдите в **Deployments** и нажмите **Redeploy** на последнем деплое

### 4. Для локальной разработки:
   - Создайте файл `.env.local` в корне проекта (НЕ коммитьте его в git!)
   - Добавьте переменные:
     ```
     UPLOADTHING_TOKEN=ваш_секретный_ключ_здесь
     UPLOADTHING_APP_ID=ваш_app_id_здесь
     ```
   - Запустите: `npm run dev`

## Важно:

- ⚠️ **НЕ коммитьте** `.env.local` в git (он уже в `.gitignore`)
- ⚠️ **НЕ публикуйте** секретные ключи в открытом доступе
- UploadThing бесплатно предоставляет 2GB хранилища
- Файлы загружаются напрямую в облако, минуя сервер
- URL файлов сохраняются в базе данных
