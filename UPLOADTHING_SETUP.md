# Настройка UploadThing

## Шаги для настройки:

1. **Создайте аккаунт на UploadThing:**
   - Перейдите на https://uploadthing.com
   - Зарегистрируйтесь или войдите в аккаунт

2. **Создайте новое приложение:**
   - В дашборде UploadThing создайте новое приложение
   - Скопируйте `UPLOADTHING_SECRET` и `UPLOADTHING_APP_ID`

3. **Добавьте переменные окружения в Vercel:**
   - Перейдите в настройки проекта на Vercel
   - Добавьте следующие переменные окружения:
     - `UPLOADTHING_SECRET` - ваш секретный ключ
     - `UPLOADTHING_APP_ID` - ID вашего приложения

4. **Для локальной разработки:**
   - Создайте файл `.env.local` в корне проекта
   - Добавьте те же переменные:
     ```
     UPLOADTHING_SECRET=your_secret_here
     UPLOADTHING_APP_ID=your_app_id_here
     ```

5. **Перезапустите приложение:**
   - После добавления переменных окружения перезапустите приложение на Vercel
   - Для локальной разработки: `npm run dev`

## Примечания:

- UploadThing бесплатно предоставляет 2GB хранилища
- Файлы загружаются напрямую в облако, минуя сервер
- URL файлов сохраняются в базе данных
