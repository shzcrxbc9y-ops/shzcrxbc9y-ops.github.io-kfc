# Как найти DATABASE_URL в Neon

## Пошаговая инструкция:

### 1. Войдите в ваш проект Neon
- Откройте https://neon.tech
- Войдите в аккаунт
- Выберите проект `kfc.1` (или как вы его назвали)

### 2. Найдите раздел "Подключитесь к своей базе данных"
На главной странице проекта вы увидите раздел с:
- Ветвь: Главная
- База данных: neondb
- Роль: neondb_owner
- Кнопка "Сбросить пароль"
- **"Строка соединения"** ← НАЖМИТЕ СЮДА!

### 3. Нажмите на "Строка соединения"
- Нажмите на текст **"Строка соединения"** или на иконку рядом
- Откроется окно/выпадающий список с полной строкой подключения
- Там будет строка вида:
  ```
  postgresql://neondb_owner:ВАШ_ПАРОЛЬ@ep-xxx-xxx.us-east-1.aws.neon.tech/neondb?sslmode=require
  ```
- Нажмите кнопку **"Copy"** или **"Скопировать"** чтобы скопировать

**Вариант B: В настройках проекта**
1. В левом меню нажмите на "Dashboard" или "Главная"
2. Найдите секцию "Connection Details" 
3. Там будет строка вида:
   ```
   postgresql://username:password@ep-xxx-xxx.region.aws.neon.tech/dbname?sslmode=require
   ```
4. Нажмите кнопку "Copy" или "Скопировать"

**Вариант C: Через Settings**
1. В левом меню нажмите "Settings" или "Настройки"
2. Перейдите в раздел "Connection string" или "Строка подключения"
3. Скопируйте строку подключения

### 3. Формат DATABASE_URL
Строка будет выглядеть примерно так:
```
postgresql://username:password@ep-cool-darkness-123456.us-east-1.aws.neon.tech/neondb?sslmode=require
```

### 4. Если не видите пароль
Если видите только часть строки без пароля:
1. В разделе "Connection Details" найдите "Password" или "Пароль"
2. Нажмите "Show" или "Показать" чтобы увидеть пароль
3. Или нажмите "Reset password" чтобы создать новый пароль
4. Затем скопируйте полную строку подключения

### 5. Альтернативный способ - создать новую строку подключения
1. В настройках проекта найдите "Connection pooling" или "Пул подключений"
2. Выберите "Session mode" или "Transaction mode"
3. Скопируйте новую строку подключения

## Важно:
- **Не делитесь** этой строкой публично - она содержит пароль!
- Используйте эту строку только в переменных окружения Vercel
- После копирования сразу добавьте в Vercel Environment Variables
